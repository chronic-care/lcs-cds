Define “55 through 80”
Patient is currently 55 through 80 years old based on date of birth

Define “Is current smoker”
Observation where code = LOINC 72166-2, Tobacco smoking status
valueCodeableConcept where code in:
http://snomed.info/sct 449868002 Current every day smoker
http://snomed.info/sct 428041000124106 Current some day smoker
http://snomed.info/sct 428071000124103 Current Heavy tobacco smoker
http://snomed.info/sct 428061000124105 Current Light day smoker

Define “Is former smoker who quit within past 15 years”
Observation where code = LOINC 72166-2, Tobacco smoking status
valueCodeableConcept where code in:
http://snomed.info/sct 8517006 Former smoker
effectivePeriod end date is < 15 years ago

Define “Has lung cancer”
Condition where clinicalStatus in ‘active’,’recurrence’, or ’relapse’
Code in VSAC https://vsac.nlm.nih.gov/valueset/2.16.840.1.113762.1.4.1116.89/expansion/Latest

Define “Had chest CT in past year”
Procedure where performedDateTime < 1 yr ago
status = completed
code = Utah-specific code set above

Main logic:
IF 55 through 80 AND (Is current smoker or Is former smoker who quit within past 15 years) AND (NOT Has lung cancer) AND (NOT Had chest CT in past year) THEN
Text recommendation: “Recommend shared decision making for lung cancer screening”
